# DescrifyAI

An AI-powered web application that generates professional e-commerce product descriptions for food businesses. Users enter product details and choose tones like premium, traditional, or health-focused to create marketplace-ready listings quickly and efficiently.

## Features

- AI-generated product descriptions
- Tone selection for premium, traditional, health-focused, and more
- Fast e-commerce listing copy creation

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open the app in the browser at the URL shown in the terminal.

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
